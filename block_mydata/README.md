# block_mydata — Mi Panel Personal

Bloque Moodle que muestra un panel personalizado al estudiante con sus estadísticas de aprendizaje: actividades, cursos, insignias, certificados y más.

## Características

- Tarjetas de estadísticas configurables (el admin elige cuáles mostrar)
- Información de perfil del usuario con avatar
- Colores personalizables por tarjeta desde la administración
- Compatible con Moodle 4.1+
- Bilingüe: español / inglés

## Instalación

1. Descomprimir el zip en `/blocks/mydata/` dentro de tu Moodle
2. Ir a Administración del sitio → Notificaciones para instalar
3. Agregar el bloque al Área personal del usuario
