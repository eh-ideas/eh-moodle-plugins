# Bloque de Moodle: Mi Perfil
El bloque "Mi Perfil" es un bloque de Moodle diseñado para mostrar de manera organizada y visualmente atractiva seis tarjetas con información relevante para el usuario.

## Tarjetas Disponibles:
- **Actividades Pendientes:** Muestra la cantidad de actividades que el usuario tiene pendientes de completar.
- **Actividades Completadas:** Indica cuántas actividades ha completado el usuario hasta el momento.
- **Mensajes sin Leer**: Informa al usuario sobre la cantidad de mensajes nuevos que aún no ha leído.
- **Insignias Recibidas:** Muestra las insignias que el usuario ha recibido por completar cursos o actividades.
- **Certificados Recibidos:** Lista los certificados que el usuario ha obtenido por finalizar cursos o lograr ciertos objetivos.
- **Cursos Completados:** Muestra la cantidad de cursos que el usuario ha finalizado con éxito.

## Compatibilidad:
El bloque "Mi Perfil" es compatible con Moodle 4.0 y versiones posteriores. Se recomienda utilizar la última versión de Moodle para garantizar la compatibilidad con las últimas características y mejoras.

## Dependencias:
El bloque está diseñado para integrarse perfectamente con la plantilla New Learning. Además, requiere que el plugin de customcerticate esté instalado para mostrar los certificados. Se recomienda actualizar este plugin para garantizar su compatibilidad.

## Ajustes del Bloque:
Para configurar el bloque "Mi Perfil", vaya a `Administración > Extensiones > Bloques > Mi Perfil` en su instancia de Moodle. Desde allí, puede personalizar los siguientes aspectos:

## Colores:
Cambie el color del texto y el fondo de cada tarjeta individualmente utilizando códigos HEX para adaptarse al diseño de su sitio.

## Iconos:
Se utilizó la librería simple-line-icons 2.4.0 para los iconos. Consulte la documentación de la librería para cambiar los iconos según sea necesario.

## Instalación:
1. Descargue el archivo ZIP del repositorio.
2. Descomprima el archivo y copie la carpeta moodle-block_mydata en el directorio blocks de su instalación de Moodle.
3. Actualice su sitio de Moodle para que reconozca el nuevo bloque.
4. Configure los ajustes según las necesidades de su institución en `Administración > Extensiones > Bloques > Mi Perfil.`
