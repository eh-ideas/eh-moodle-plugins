# block_academichistory

Plugin para Moodle que proporciona funcionalidades específicas.

## Descripción

Componente personalizado para Moodle 4.x

## Requisitos

- Moodle 4.0 o superior
- PHP 7.4 o superior

## Instalación

1. Descargar el plugin
2. Extraer en la carpeta correspondiente de Moodle
3. Visitar Administración > Sitio > Notificaciones para completar la instalación

- Funcionalidades específicas del plugin
